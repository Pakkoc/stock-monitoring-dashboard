# 업무 내용

## 프로젝트 개요

- 주식 정보 모니터링 대시보드 구축

## 프로젝트 배경 및 목표

- 현재 개인 주식 투자를 진행하면서, 여러 웹사이트와 HTS에 분산된 정보를 확인하는 데 불편함을 겪고 있습니다. 다양한 서비스에서 제공하는 유용한 기능들(예: 조건별 종목 정렬, 테마별 그룹핑, 관련 뉴스 확인)을 하나의 화면에서 통합적으로 조회할 수 있는 개인 맞춤형 대시보드를 개발하고자 합니다. 본 프로젝트의 목표는 HTS와 함께 보조 모니터에 띄워놓고 사용할 수 있는 웹 기반의 주식 현황판을 구축하여, 분산된 정보를 통합하고 급등 종목의 발생 원인(뉴스 등)을 신속하게 파악함으로써 보다 빠르고 효율적인 투자 의사결정을 지원하는 것입니다.

## 과업 범위

### 1. 수행 범위

- 상세 기획: 요구 사항 정의, 기능 정의, 화면 설계
- UI/UX 디자인
- 프런트엔드/Client 개발 (PC Web)
- 백엔드 개발
- 서버/DB/인프라 구성

### 2. 상세 기능 요구 사항

#### 2-1. 핵심: 주식 정보 대시보드

- 사용자가 원하는 종목들을 등록하고 주요 정보를 한눈에 볼 수 있는 위젯 기반의 대시보드 화면을 제공해야 합니다.

#### 2-2. 다양한 조건별 종목 정렬 및 필터링

- 전체 종목 또는 관심 종목 리스트를 거래대금, 등락률 등 다양한 기준으로 실시간 정렬 및 필터링할 수 있는 기능을 제공해야 합니다.

#### 2-3. 테마별 종목 그룹핑

- 특정 테마에 속한 종목들을 그룹으로 묶어서 관리하고 조회할 수 있는 기능을 제공해야 합니다.

#### 2-4. 종목 연관 뉴스 피드

- 특정 종목을 선택했을 때, 해당 종목과 관련된 최신 뉴스를 시간순으로 리스팅하여 제공해야 합니다.

#### 2-5. 관리자 기능

- 대시보드 기본 설정, 데이터 연동 API 키 관리 등 기본적인 관리 기능을 포함해야 합니다.

## 지원 디바이스 및 디자인

### 1. 지원 디바이스

- PC Web (데스크톱 환경 최적화)

### 2. 디자인 가이드

- 벤치마킹 대상: 프로젝트 진행 시, 클라이언트가 선호하는 다수의 레퍼런스 사이트 및 서비스를 제공할 예정입니다.

## 기술 스택

- 프로젝트에 가장 적합하다고 판단되는 기술 스택을 자유롭게 제안해 주시기 바랍니다.

## 지원 자격 및 우대 사항

### 1. 지원 자격

- 주식 관련 서비스(HTS, MTS, 정보 사이트) 개발 경험

### 2. 우대 사항

- 실시간 데이터 연동 및 처리 경험
- 데이터 시각화 및 대시보드 구축 프로젝트 경험
