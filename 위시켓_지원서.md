[프로젝트 진행 제안]

안녕하세요. 요구사항을 꼼꼼히 분석한 후 지원드립니다.
주식 모니터링 대시보드의 5가지 핵심 기능(위젯 대시보드, 조건별 정렬/필터, 테마 그룹핑, 종목 뉴스 피드, 관리자 기능)을 모두 구현 가능하며, 현재 동작하는 포트폴리오를 보유하고 있어 미팅 시 실제 시연이 가능합니다.

개발 일정 (총 30일)

Phase 0: 기획 및 설계 (1~5일차)
- 요구사항 상세 분석 및 기능 명세서 작성
- 화면 설계서 작성 (Figma 기반 와이어프레임)
- 클라이언트 제공 레퍼런스 사이트 분석 및 디자인 방향 수립
- DB 스키마 설계 (종목, 시세, 뉴스, 테마, 관심종목)
- API 설계서 (REST 엔드포인트 30개+ 정의)
- 클라이언트 확인 및 피드백 반영

Phase 1: UI/UX 디자인 + 인프라 구축 (6~12일차)
- UI/UX 디자인 (Figma — 대시보드, 종목 상세, 관리자 페이지)
- 클라이언트 레퍼런스 기반 디자인 시안 작성 및 피드백
- 프로젝트 스캐폴딩 (모노레포, Docker, CI/CD)
- 증권사 API 연동 (실시간 시세 수신 파이프라인)
- 데이터베이스 구축 (PostgreSQL + 시계열 DB)
- 인증/인가 시스템 구현

Phase 2: 핵심 기능 개발 (13~22일차)
- 위젯 기반 대시보드 (드래그앤드롭, 리사이즈, 레이아웃 저장)
- 조건별 종목 정렬/필터링 (거래대금, 등락률, 거래량 등 다중 조건)
- 테마별 종목 그룹핑 + 테마 성과 분석
- 종목 연관 뉴스 피드 (다중 소스 통합, 종목-뉴스 자동 매칭)
- 급등 종목 감지 + 원인 자동 분석 (뉴스/테마/수급 분류)
- 캔들스틱 차트 (1분/5분/15분/1시간/일봉)
- 관심종목 관리 (추가/제거, 실시간 가격 표시)

Phase 3: 관리자 기능 + 품질 검증 (23~28일차)
- 관리자 기능 (대시보드 기본 설정, API 키 관리, 데이터 수집 모니터링)
- 통합 테스트 및 버그 수정
- 성능 최적화 (2,500+ 종목 대응)
- 프로덕션 배포 및 운영 환경 구성

Phase 4: 안정화 및 인수인계 (29~30일차)
- 실 사용 테스트 및 피드백 반영
- 운영 문서 및 인수인계

미팅 시 협의 희망 사항
- 벤치마킹 레퍼런스 사이트 확인 (디자인 시안 작업에 필요)

- 급등 감지 임계값 기준 (기본 5% 제안, 조정 가능)
- 배포 환경 (클라이언트 서버 또는 클라우드 배포 방식)


[유사 프로젝트 진행 경험]

암호화폐 실시간 아비트라지 모니터링 대시보드 (개인 프로젝트)

본 프로젝트와 기술적으로 매우 유사한 실시간 금융 데이터 대시보드를 직접 설계/개발한 경험이 있습니다.

프로젝트 개요
- 5개 거래소(빗썸, 업비트, 코인원, 바이낸스, 바이빗)의 BTC/ETH 실시간 시세를 WebSocket으로 수신
- 거래소 간 가격 차이(스프레드) 및 김치 프리미엄을 실시간 계산
- 임계값 초과 시 텔레그램 자동 알림 발송
- React + Zustand + Recharts 기반 실시간 대시보드

본 프로젝트와의 공통점
- 실시간 금융 데이터 수신 파이프라인 (WebSocket/폴링 -> 캐시 -> 프론트엔드)
- 자동 재연결 + 지수 백오프 (1초 -> 60초 상한) 기반 안정적 데이터 수신
- 임계값 기반 알림 시스템 (급등 감지와 동일한 구조)
- Zustand 상태관리 + TanStack Query 서버 상태 조합
- 실시간 차트 렌더링 (Recharts 시계열 차트)
- SQLite/PostgreSQL 시계열 데이터 저장 + 보존 정책

핵심 기술 역량
- 거래소 API 연동: OAuth2 인증, Rate Limit 관리, 6-state FSM 기반 연결 관리
- 실시간 데이터 처리: 5개 거래소 동시 WebSocket 스트리밍 + 10초 간격 DB 스냅샷
- 알림 엔진: 3단계 심각도 (INFO/WARNING/CRITICAL) 자동 분류 + 텔레그램 발송
- 대시보드: React + WebSocket 실시간 바인딩, 가격/스프레드/알림 동시 표시

포트폴리오 시연 가능
- 미팅 시 실제 동작하는 시스템을 시연할 수 있습니다.


[사용 기술과 툴]

기술 스택
- 프론트엔드: Next.js, React, TypeScript, Tailwind CSS

- CI/CD: GitHub Actions
- 디자인: Figma
